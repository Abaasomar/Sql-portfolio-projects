## 1. Project Overview

This project analyzes retail transaction data from multiple grocery stores to understand:

- **Sales performance:** Which products and stores generate the most revenue.
- **Discount strategies:** How stores use promotions and their effectiveness.
- **Customer loyalty:** Which products drive repeat purchases.
- **Product diversity:** Variety of products overall and per aisle.
- **Store performance:** Geographic coverage and top-performing locations.

**Tools Used:**

- MySQL / SQL Workbench

**Database:**

- `sql_portfolio_projects`
- Main table: `grocery_store2`
