## Projects Included

This repository contains the following projects:

1. **Somalia Financial Journey (`somalia_financial_j`)**  
   - Focus: Analyzing Somalia government financial data to understand revenue, expenditures, and trends.  
   - Key Components:
     - SQL scripts for database creation and financial data exploration.
     - Queries to calculate annual spending, ministry-level expenditures, cost type analysis, and international project allocations.
     - Insights and recommendations for improving budget allocation and financial transparency.

2. **Grocery Store Sales Analysis (`grocery_store_sales`)**  
   - Focus: Analyzing grocery store transactions to uncover sales performance, discount strategies, customer loyalty, and product diversity.  
   - Key Components:
     - Data cleaning SQL scripts.
     - SQL queries for revenue, discount penetration, loyalty program evaluation, and aisle-level product analysis.
     - Insights and actionable recommendations to optimize sales, promotions, and store operations.

3. **Supplement Sales Analysis (`supplement_sales`)**  
   - Focus: EDA on supplement sales data to understand revenue performance, discounts, platform performance, and geographic trends.  
   - Key Components:
     - Data cleaning and preparation SQL scripts.
     - SQL queries exploring revenue by category, platform, country, and year-over-year trends.
     - Insights highlighting top-performing categories and platforms.
     - Recommendations for marketing, inventory, and platform strategy optimization.

4. **E-commerce Transactions Analysis (`ecommerce_transactions`)**  
   - Focus: Analyzing e-commerce transactions to uncover revenue patterns, discount strategies, sales trends, and customer payment behavior.  
   - Key Components:
     - Data cleaning SQL scripts (`01_data_cleaning.sql`) for missing discounts, date standardization, and logical checks.
     - SQL queries (`02_exploratory_data_analysis.sql`) for EDA: revenue per category, sales trends over time, discounts per category, and payment behavior.
     - Insights (`insights/insights.md`) summarizing key observations and trends.
     - Recommendations (`insights/recommendations.md`) with actionable business strategies.
